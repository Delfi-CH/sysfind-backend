# sysfind-backend

application backend for managing operating system (installers)

## Docker

```bash
    git clone https://github.com/Delfi-CH/sysfind-backend.git

    cd sysfind-backend

    docker compose up
```

## Build from source

```bash
git clone https://github.com/Delfi-CH/sysfind-backend.git

cd sysfind-backend

npm install

npm run dev
```
